# 1. DOM Assignment 1

### Task 1
>Add an element named "Hire Me" in the navigation menu.

**After Update**
![Task 1 Output](./task1Output.png)


```javascript

// Solution of Task:1

const navList = document.querySelector("nav ul");
const element = document.createElement("li");
element.innerHTML = "<a>Hire Me</a>";
navList.appendChild(element);
```
